vue项目模版仓库

## 简介

此模版没有任何预设，只是提供一个最小的运行页面。
 

## 使用

```bash
# 需要node版本>= 12.0.0, degit的方式不会对仓库产生git log info

npx degit git@github.com:lvfangren/WYSIWYG.git#main u-project-name

cd u-project-name

# git config
git init
git remote set-url origin XXX(u self repositories)

npm install

npm run dev

```
